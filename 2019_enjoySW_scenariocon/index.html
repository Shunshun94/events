ソードワールドを遊びつくそう！
初心者用シナリオコンテスト

1. はじめに

SWプレイヤーの皆様初めまして。そして、「ソード・ワールド2.5」始動おめでとうございます。2.5からプレイヤーとなった方々はようこそ。

さて、コンテストを開催するにあたって"初心者用シナリオ"と条件付けました。私達は SW RPG をはじめたてだった時分を振り返ったのです。手探りでソード・ワールドという TRPG に踏み込み、魅入られた頃。試行錯誤ゆえの楽しみもありました。

しかし、今思えば手の届かない痒い所が多くあったのも事実です。戦闘バランスの調整然り、場面転換の方法然り。挙げればキリがありません。"あの頃" に感じた歯がゆさを活かし、ソード・ワールドへの新たなる「冒険者」達をラクシアへ送り出そう、というのが本イベントの着想です。

皆様のシナリオがこれからPLとして、あるいはGMとして門を叩く者の道しるべとならんことを願っています。

3. エントリー方法

募集開始～募集終了の期間にTRPGのシナリオ投稿＆検索サイトscenarch（シナーチ）にシナリオを投稿してください。また、投稿したシナリオには「第一回SWシナリオコン」のタグを設定してください。投稿とタグ付けをもってエントリー完了となります

シナリオの公開場所は任意とします。しかし、scenarch への登録とタグの設定はエントリーに必須です

エントリーに際して、他の申し込みは不要です

4. 投稿ルール

  1. ソード・ワールド2.0またはソード・ワールド2.5のシナリオを作成し、投稿してください
  2. シナリオは初心者 GM がマスタリングしやすく、初心者 PL がプレイしやすいものを作成してください
  3. 1人いくつでも投稿が可能です
  4. シナリオは投稿者が著作権等の必要な知的財産権を持つものを投稿してください

禁止事項

他者が作ったシナリオの許諾無しでの投稿

ルールブックなしでプレイできるように書かれたシナリオの投稿

シナリオの記述方法について

文字数や記述形式について制限は設けません。図表の利用も制限されません。初心者向きの分量、記述方法を意識して記述してください

シナリオのレギュレーション

  1. 必須ルールブックは以下の通りとしてください。つまり、どちらの場合も最低限ルールブック1と2があればプレイできる状態にしてください

    ソード・ワールド2.0ソード・ワールド2.0 ルールブック1 改訂版
    ソード・ワールド2.0 ルールブック2 改訂版
    ソード・ワールド2.5ソード・ワールド2.5 ルールブック1
    ソード・ワールド2.5 ルールブック2

  2. プレイヤーキャラクターを作成するときのレギュレーションは上述のルールブックの範囲で任意とします
  3. プレイ時間について、原則として1シナリオ8時間を上回らない分量としてください
    (オンラインテキストセッションの場合を基準とします)

シナリオの権利

  • 投稿したシナリオの著作権などは作者に帰属します
  • 第三者が投稿したシナリオを元にした動画やリプレイの公開する場合、使用条件を明記することを強く推奨します
  • コンテストの主催は投稿されたシナリオを SNS 上でシェアすることがあります。この際、シナリオが投稿者の著作物である旨がシェアされる投稿には明記されます

どのようなシナリオがよいのか

GM も PL も初心者であったとしても使いやすいシナリオであることが望まれます

例

  • 立卓が容易なシナリオである
  • GM にとって読みやすく、プレイしやすいシナリオである
  • PL にとって導入されやすいシナリオである
  • 自作シナリオの参考にしやすいシナリオである

また、さらに投稿者が考えるさらなる初心者向け要素を付加することも期待されます
