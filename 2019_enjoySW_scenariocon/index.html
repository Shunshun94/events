ソードワールドを遊びつくそう！
初心者用シナリオコンテスト

1. はじめに

SWプレイヤーの皆様初めまして。そして、「ソード・ワールド2.5」始動おめでとうございます。2.5からプレイヤーとなった方々はようこそ。

さて、コンテストを開催するにあたって"初心者用シナリオ"と条件付けました。私達は SW RPG をはじめたてだった時分を振り返ったのです。手探りでソード・ワールドという TRPG に踏み込み、魅入られた頃。試行錯誤ゆえの楽しみもありました。

しかし、今思えば手の届かない痒い所が多くあったのも事実です。戦闘バランスの調整然り、場面転換の方法然り。挙げればキリがありません。"あの頃" に感じた歯がゆさを活かし、ソード・ワールドへの新たなる「冒険者」達をラクシアへ送り出そう、というのが本イベントの着想です。

皆様のシナリオがこれからPLとして、あるいはGMとして門を叩く者の道しるべとならんことを願っています。

3. エントリー方法

TRPGのシナリオ投稿＆検索サイトscenarch（シナーチ）にて応募作品に「第一回SWシナリオコン」のタグを付けてエントリー完了です。

エントリーに申し込みは不要です。

シナリオの公開場所は問いませんがシナーチへの作品登録を別途に行う必要があります。

4. 投稿ルール

  1. 応募作品がソード・ワールド2.0またはソード・ワールド2.5のシナリオであること
  2. 初心者 GM がマスタリングしやすく、初心者 PL がプレイしやすいものを心がけること
  3. 【ソード・ワールド】二次創作ガイドライン (https://fujimi-trpg-online.jp/info/entry-2587.html) に則ること
  4. 投稿者が著作権等の必要な知的財産権を持つものを投稿すること
  5. 投稿数に上限はない

禁止事項

制作者の許諾なしの既存の改変シナリオの投稿

※不適とされた応募作品は審査対象になりません

シナリオの記述方法について

文字数や記述形式について制限は設けません。図表の利用も制限されません。初心者向きの分量、記述方法を意識して記述してください

シナリオのレギュレーション

  1. プレイヤーキャラクター作成時の規定は下記のルールブックの範囲で任意とする。
  2. プレイ時間について、原則として1シナリオ8時間を上回らない分量とする。(テキストセッション基準)
  3. 文字数や記述形式について制限はなし。図表の利用についても同様。
  4. 下記システムのシナリオであることとする。各システムはルールブックの2までを用いることができる。1のみでも構わない。
    • ソード・ワールド2.0
    • ソード・ワールド2.5

シナリオの権利

  • 投稿したシナリオの著作権などは作者に帰属する
  • 第三者が投稿したシナリオを元にした動画やリプレイの公開する場合、使用条件を明記することを強く推奨する
  • コンテストの主催は投稿されたシナリオを SNS 上でシェアすることがある。この際、シナリオが投稿者の著作物である旨がシェアされる投稿には明記される
